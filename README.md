# Cebufreelancer
An online marketplace for freelancers and employeers. Built using CodeIgniter & MySQL.

### Features
* User Login & Registration   
* Post a Project   
* Bid a Project   
* Create Freelancer Profile   
* Report a Scam   
* Send Private Message to the Buyer   
* Tell a Friend
